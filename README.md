# readme-wordcloud

> generate wordcloud for your `README.md` file.

## Usage

inject HTML syntax comment in your `README.md` like:

```html
generate the dependenies and devDependenies wordcloud
<!-- cloud:start-->
<!-- cloud:end -->
```
